# 🧮 Calculator App

A simple and responsive calculator web application built using HTML, CSS, and JavaScript. This app allows users to perform basic arithmetic operations with a clean and user-friendly interface.

---

## 📌 Features

- ➕ Addition  
- ➖ Subtraction  
- ✖️ Multiplication  
- ➗ Division  
- 📊 Percentage calculation  
- 🧹 Clear all input (C / CE)   
- 🌙 Light/Dark mode support  

---

## 🛠️ Tech Stack

- **HTML5** – Structure of the application  
- **CSS3 / Tailwind CSS** – Styling and responsiveness  
- **JavaScript (ES6+)** – Logic and event handling  
- **Math.js** – Evaluating mathematical expressions  

---

## 🚀 How It Works

1. User clicks numbers and operators  
2. Input is displayed in the input box  
3. On clicking `=`:
   - Expression is evaluated using **Math.js**  
4. If no operator is present:
   - An error message is displayed

## ⚙️ Setup Instructions

1. Clone the repository  
```bash
git clone <your-repo-link>
```

2. Navigate to the project folder  
```bash
cd calculator-app
```

3. Install dependencies  
```bash
npm install
```

4. Run the development server  
```bash
npm run dev
```
